# lampa-3gubki

Форк медиацентра [Lampa](https://github.com/yumata/lampa-source). Оставлен только веб-интерфейс — всё лишнее вырезано, добавлен свой функционал для работы с торрентами.

Витрина TMDB (фильмы / сериалы / мультфильмы), поиск раздач через Jackett или Prowlarr, воспроизведение через TorrServer и внешний плеер, интеграция с qBittorrent.

## Возможности

- **Торренты** — поиск и выбор раздачи в модальном окне поверх страницы; своя логика для фильмов и сериалов (выбор качества, для сериалов — выбор сезона).
- **qBittorrent** — отправка раздач на загрузку и страница «Загрузки» для просмотра и управления скачанным.
- **Поиск** — компактное всплывающее окно.
- **Интерфейс** — переработанные меню и модальные окна.

## В планах

- Раздел «Аниме» — отдельная логика поиска и подбора раздач.

## Запуск

```bash
docker compose up -d --build
```

Откроется на `http://<адрес-сервера>:8095`.
