# lampa-3gubki

Персональный форк медиацентра [Lampa](https://github.com/yumata/lampa-source) для домашнего сервера: от оригинала оставлен только веб-интерфейс, а работа с торрентами переписана под свои задачи.

Lampa — это витрина фильмов и сериалов по данным TMDB с поиском раздач и воспроизведением через TorrServer. Здесь убрано всё лишнее, а поиск и выбор торрентов вынесен в пошаговые модальные окна с отдельной логикой под каждый тип контента.

## Возможности

- **Торренты.** Раздача выбирается в модальном окне поверх страницы, без перехода на отдельный экран. Для фильмов и сериалов — своя логика: подбор по качеству (4K HDR / 4K / 1080p / 720p), для сериалов дополнительно выбор сезона. Мусорные раздачи отсеиваются, остальные сортируются под выбранное качество.
- **qBittorrent.** Любую раздачу можно отправить на загрузку в qBittorrent. Скачанное собрано на странице «Загрузки» — с постерами из TMDB, навигацией по папкам, воспроизведением и управлением активными раздачами.
- **Поиск.** Компактное всплывающее окно вместо полноэкранной страницы: результаты TMDB появляются сразу под строкой ввода.
- **Интерфейс.** Переработаны боковое меню и модальные окна.

## В планах

Отдельный раздел для аниме — с собственной логикой поиска и подбора раздач.

## Запуск

```bash
docker compose up -d --build
```

Интерфейс открывается на `http://<адрес-сервера>:8095`.
