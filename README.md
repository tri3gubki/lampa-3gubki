# lampa-3gubki

Персональный форк [Lampa](https://github.com/yumata/lampa-source) — медиацентр для дома, собранный из исходников с кастомизацией под собственный сервер.

## Что переделано относительно upstream

**qBittorrent-интеграция**
- Отправка торрентов в qBittorrent прямо из списка раздач (long-press → «Скачать в qBittorrent»), с приоритетом первого/последнего куска и последовательной загрузкой.
- Раздел настроек qBittorrent (адрес Web UI, токен удаления).
- Страница «Загрузки» — просмотр скачанных файлов с сервера: постеры/стоп-кадры из TMDB, навигация по папкам, воспроизведение прямо с сервера, удаление через qBit API + WebDAV, play/pause активных раздач.

**Поиск**
- Компактный попап вместо полноэкранной страницы. TMDB multi-search (фильмы + сериалы вперемешку), результаты в одном горизонтальном ряду.

**Торренты**
- Список открывается модальным окном поверх страницы фильма.
- Для фильмов — фиксированный поиск «оригинал + год», фильтр мусора (только 4K/1080p/720p, живые раздачи, без камрипов, размер ≥ 3 ГБ), дедуп дублей по сигнатуре.
- Одиночный файл-фильм запускается сразу, без окна выбора.

**Интерфейс**
- Левое меню — вертикальная компоновка (иконка над текстом), пункт «Загрузки».
- Верхний трей: поиск / каталог / настройки / выход.
- Модальные окна — единый тёмный полупрозрачный стиль, fade-анимация.
- Убраны рекомендации/похожие на странице фильма, секция тегов, мёртвые модули CUB-наследия.

## Запуск

```bash
docker compose up -d --build
```

Откроется на `http://<server>:8095`.

Сборка многоступенчатая: `node:20-alpine` собирает Lampa из `lampa-source/` через gulp (rollup + SCSS + языки), результат раздаётся через `nginx:alpine`.

## Структура

```
lampa-source/      исходники Lampa (форк yumata/lampa-source) с правками в src/
Dockerfile         multi-stage сборка
docker-compose.yml сервис lampa-web (порт 8095)
```

## qBittorrent — настройка сервера

Для работы раздела «Загрузки» и отправки торрентов нужен nginx-прокси перед qBittorrent:
- `/files/` — раздача скачанных файлов (autoindex JSON + Range + WebDAV DELETE).
- `/api/v2/` — проксирование qBit Web UI API.
- CORS-заголовки для домена Lampa.
- qBittorrent: bypass-авторизация для локальной сети.

Адрес прокси указывается в Lampa: Настройки → qBittorrent.
