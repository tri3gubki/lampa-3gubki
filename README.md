# lampa-3gubki

Персональный форк [Lampa](https://github.com/yumata/lampa-source) — домашний медиацентр, собранный из исходников с переработкой интерфейса и торрент-флоу под собственный сервер.

Витрина TMDB (фильмы / сериалы / мультфильмы), поиск раздач через Jackett или Prowlarr, воспроизведение через TorrServer или внешний плеер, интеграция с qBittorrent.

## Возможности

### Торренты — пошаговые окна

Вместо одного общего списка — модальное окно поверх страницы, своё под каждый тип контента.

- **Фильмы:** «Смотреть» → поиск в фоне → выбор качества (4K HDR / 4K / 1080p / 720p) → топ-5 раздач → плеер.
- **Сериалы:** «Смотреть» → выбор сезона → поиск именно этого сезона → выбор качества → топ-5 раздач → окно выбора серий → плеер.

Фильтрация раздач:

- отсев мусора — камрипы, раздачи без распознанного качества;
- для сериалов — точный сезон по блоку `S..E..`, сверка года сезона (отсекает одноимённые шоу и спин-оффы), многосезонные сборники как запасной вариант;
- сортировка: 4K — по числу раздающих, 1080p — по битрейту (размеру);
- 720p показывается только когда нет качества выше.

Долгое нажатие на раздаче → меню «Смотреть» / «Скачать в qBittorrent».

### qBittorrent

- Отправка раздач в qBittorrent прямо из списка торрентов.
- Раздел настроек (адрес Web UI, токен удаления).
- Страница **«Загрузки»** — файловый менеджер скачанного с сервера: постеры и стоп-кадры серий из TMDB, навигация по папкам, воспроизведение с сервера, удаление, пауза/запуск активных раздач.

### Поиск

Компактное всплывающее окно вместо полноэкранной страницы. Поиск по TMDB — фильмы и сериалы вперемешку в одном ряду, нативная клавиатура устройства.

### Интерфейс

- Левое меню — вертикальная компоновка (иконка над подписью).
- Верхняя панель: поиск / каталог / настройки / выход.
- Модальные окна — единый тёмный полупрозрачный стиль.
- Убраны рекомендации/похожие, теги, неиспользуемые модули.

## Запуск

```bash
docker compose up -d --build
```

Откроется на `http://<адрес-сервера>:8095`.

Сборка многоступенчатая: `node:20-alpine` собирает Lampa из `lampa-source/` (gulp → rollup + SCSS + языки), готовая статика раздаётся через `nginx:alpine`.

## Структура

```
lampa-source/       исходники Lampa (форк yumata/lampa-source), правки в src/
Dockerfile          сборка
docker-compose.yml  сервис lampa-web (порт 8095)
```

## qBittorrent — настройка

Для раздела «Загрузки» и отправки раздач нужен nginx-прокси перед qBittorrent:

- `/files/` — раздача скачанных файлов (листинг JSON + Range + WebDAV DELETE);
- `/api/v2/` — проксирование qBittorrent Web API;
- CORS-заголовки для адреса Lampa;
- в qBittorrent — обход авторизации для локальной сети.

Адрес прокси указывается в Lampa: Настройки → qBittorrent.
